# Readly

## Client Side
The front end is a Chrome extension.

Install the folder `readly-chrome` as an unpacked extension in Chrome.

## Server Side
Server side is using `FastAPI` (python) to run an HTTPS service

```
python -m spacy download en_core_web_sm
```
